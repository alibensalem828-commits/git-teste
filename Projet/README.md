VS code sur une tablette 

Comment j'ai transformé ma tablette Android en véritable environnement de développement, avec VS Code accessible dans le navigateur, sans PC et sans root.
MON PROBLEME???:
je voulais coder depuis ma tablette android ou telephone comme sur un vrai ordinateur et avoir un vrai editeur (vs code),un terminal linux,et  installer node.js,javascript et python (et autres mais vous avez compris)
en plus ,J'avais pas de pc portable donc raison de plus 
LA SOLUTION!!!:
Faire tourne code-server (le vs code officiel mais accessible via le navigateur ) a l'interieur d'un linux qui est dans un terminal(Termux), puis y accéder en tapant une adresse IP dans le navigateur de la tablette.

Résultat : VS Code s'ouvre dans le navigateur, avec toutes ses fonctionnalités.
